Projects

Scheme Parser and Evaluator

Created a lexer and parser for the functional programming language 'Scheme.' Used this output in a metacircular evaluator for the same language that allowed for programs to be run with the additional distinction of static and dynamic scope.

Processor Simulator

Euchre

Designed a playable version of the card game 'Euchre.' The game uses a slightly modified ruleset and involves both human and simple AI players.
